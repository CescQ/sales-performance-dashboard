Sales Performance Dashboard
SQL Server + Power BI Demo Project

Overview:
This project showcases a Sales Performance Dashboard built using SQL Server for data storage and Power BI for visualization.

Features:
Total Sales KPI
Orders KPI
Total Quantity KPI
Average Order Value (AOV)
Sales Trend Over Time (Line Chart)
Sales by Product Category (Column + Pie Chart)
Sales by Customer Company
Interactive slicers:
Product Category
Customer Country

SQL Server
Power BI
